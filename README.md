# EVE PI Manager

[Deutsch](README.de.md) | [English](README.en.md) | [简体中文](README.zh-Hans.md)

Self-hosted Planetary Industry manager for EVE Online.

If this project helps you, Ingame-ISK donations to `DrNightmare` are welcome.

## Read The Docs

- [Deutsch](README.de.md)
- [English](README.en.md)
- [简体中文](README.zh-Hans.md)

## Highlights

- Dashboard, Skyhooks, Characters, Corporation, System Analyzer, Compare, System Mix, PI Chain Planner
- DB-backed caches for market prices, dashboard values, skyhook values, and GUI translations
- GUI languages: German, English, Simplified Chinese
- Linux, Docker Compose, and native Windows setup scripts
